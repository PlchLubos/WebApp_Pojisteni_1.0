O aplikaci

Specifikace:
  • Aplikace obsahuje správu pojištěných
    • Vytvoření pojištěného
      • Eviduje jméno, příjmení, věk, telefon
    • Zobrazení seznamu všech pojištěných
  • Dané entity jsou uloženy v Local Storage
  • Obsahuje Bootstrap i JS validaci vstupního formuláře
  • Telefonní číslo je formátováno v JS do jednotného formátu
  • Aplikace je responzivní
Použité technologie:
  • HTML
  • CSS, Bootstrap 4
  • JavaScript
  • Git, GitHub
